## 📚 교재 시스템

### ORT (Oxford Reading Tree)
- **ORT1 ~ ORT10**: 각 단계별 1~6단계 × 6권 = 권당 36권
- **명명규칙**: `ORT2-3-3` = ORT2의 3단계 3번째 책

### ORT 레벨별 학습 단계 ⭐ 중요

| 레벨 | 학습 단계 |
|------|----------|
| **ORT1~2** | W·T·C·P·📓앞·D·S·Q·📔뒤·C (퀴즈까지 10단계) |
| **ORT3** | 위 + **L 라이팅 ✏️** |
| **ORT4~9** | 위 + **L 라이팅 + R ORT 리뷰 🔁** |
| **ORT10** | L 포함 (R 제외) |

### 학습 단계 약어
- **W**: 단어 (Word)
- **T**: 텍스트 / 지면 시험
- **C**: 컨셉 / 클래스카드
- **P**: 퍼즐 (Puzzle)
- **📓앞**: 워크북 앞바닥 필사
- **D**: 딕테이션 / 듣기
- **S**: 셰도잉 / 사운드
- **Q**: 퀴즈 (Quiz)
- **L**: 라이팅 (Writing) - ORT3부터
- **📔뒤**: 워크북 뒷바닥
- **R**: ORT 리뷰 - ORT4~9

## 🎫 면제권 시스템

### 핵심 로직
- 숙제 ○ **20개 달성** → **면제권 1장 자동 적립**
- 카운트는 **자동으로 0부터 다시 시작** (`oCount % 20`)
- 누적 ○ 기반 자동 변환 (`Math.floor(oCount / 20)`)

### 면제권 사용 효과
- **학생**: 사용 → 당일 숙제 카드 사라지고 "🎫 오늘 숙제 면제!" 메시지
- **선생님**: 다음 시간 과제 점검에서 자동으로 "—(면제)" 버튼만 강조
- 학생은 과제 점검 클릭 불가 (읽기 전용)

### 선생님 수동 조절
- **+ 버튼**: 보너스로 면제권 1장 추가
- **− 버튼**: 면제권 1장 차감 (실수 회수)
- **🎁 면제권 사용** 버튼: 학생 실사용 시

## 🏠 숙제 관리

### 캘린더 상태

- **×** (빨강): 미완
- **—** (회색): 면제

## 🛠️ 핵심 기능

### 단계 완료 시스템 (ORT/RG/다독)
- 완료 버튼 클릭 → 모달
- **1~8단계 점프 옵션**
- **직접 입력** 칸 (커스텀 단계/교재명)
- 입력 시 → `dadokOrder`에 자동 추가 + 카드 생성
- 완료된 카드 → **한 줄 압축** (지면 + W·P·Q 뱃지)
- 클릭하면 펼침/접힘 (`toggleCompact`)

### ORT 카드 압축 요약
- 지면 시험 + W + P + Q (4개 모두)
- RG는 지면시험만
- 다독은 W·P·Q (지면 없음)

### 시간표 설정
- 30분 단위 (14:00 ~ 19:30)
- 미수업 ↔ 시간 설정 토글 (`toggleWeekdayCell`)
- 미수업: 회색 / 시간 설정: 활성화 색상

### 보강 일정
- 헤더 + 보강 일정 관리에서 "+ 보강 추가" 버튼
- 모달: 날짜 + 시간 + 사유
- 추가 시 목록에 실제로 표시 (보라색 강조)
- ✕ 버튼으로 삭제

### 학생 데이터 모델
```javascript
const studentBooks = {
  ort: 'ORT2',
  dadok: 'Fly Guy',
  extras: ['단어800', 'RG1']
};
```
